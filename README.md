#### Запуск ESlint

- Проверка eslint

  `npm run lint`

- Проверка eslint и автофикс

  `npm run lint:fix`

#### Запуск

- Запуск проекта:

  `npm start`

#### Сборка

- Сборка проекта:

  `npm build`

#### Тесты

- Запуск теста в командной строке

  `npm run cypress:test`

- Запуск тестов в графическом интерфейсе

  `npm run cypress`
